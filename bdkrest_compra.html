MÓDULO DE VENTAS

El modulo de ventas es el principal del sistema. Aquí podrás llevar el control de tus ventas, ventas a domicilio, llevar control de las ventas de los meseros, realizar cortes de caja y muchas otras funciones.

SERVICIO A DOMICILIO

En este vídeo te haremos una demostración de como se lleva el control de las ventas con servicio a domicilio.

MÓDULO DE MESEROS

ALTA DE RECETAS

MÓDULO ADMINISTRADOR

El módulo de ADMINISTRACIÓN te sirve para dar de alta todos los catálogos del sistema como productos, categorías, formas de pago, usuarios, productos y recetas.

También sirve para ver los diferentes reportes como ventas, existencias, cortes de caja, etc.

CONCENTRADOR WEB

Módulo que permite poder monitorear de forma remota los movimientos del negocio, como las ventas, existencias, cortes de caja, etc.

* Se adquiere por separado. * Compatible con la versión Profesional y Plus
